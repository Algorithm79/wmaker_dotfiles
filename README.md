# wmaker_dotfiles
Configurationsfiles for WindowMaker


![screenshot](https://raw.githubusercontent.com/Algorithm79/wmaker_dotfiles/master/wmaker2.png)

- [Kitty](https://github.com/kovidgoyal/kitty) (Terminal)
- [Nerd Fonts](https://github.com/ryanoasis/nerd-fonts) (mononoki Nerd Font)
- [Betterlockscreen](https://github.com/pavanjadhaw/betterlockscreen)
- [Rofi](https://github.com/davatorium/rofi) (window switcher, launcher)
- [Dunst](https://github.com/dunst-project/dunst) (notification daemon)
- [Tela Circle Icons](https://github.com/vinceliuice/Tela-circle-icon-theme) (Icon theme used in Rofi)
- [Picom](https://github.com/yshui/picom) (Compositor) 
- [Starship](https://starship.rs/) (Cross-Shell Prompt)
- [Tokyo Night Gtk Theme](https://github.com/Fausto-Korpsvart/Tokyonight-GTK-Theme)
